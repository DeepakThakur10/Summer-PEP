# DOM (Document Object Model)

- DOM represents an HTML document as a tree of objects.
- JavaScript uses the DOM to access and modify HTML and CSS.
- Every HTML element is a node in the DOM tree.

Common Methods:
- document.getElementById()
- document.querySelector()
- document.querySelectorAll()

Common Properties:
- textContent
- innerHTML
- style
- value

Uses:
- Change content
- Change styles
- Add/remove elements
- Handle events
